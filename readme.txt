N&uuml;tzliche Widgets f&uuml;r deinen Reiseblog. Zeige zum Beispiel einen Reise-Countdown oder aktuelle Reiseschn&auml;ppchen in deiner Sidebar.

== Description ==

Das Plugin &quot;Reiseblog Widgets&quot; enth&auml;lt n&uuml;tzliche Sidebar-Widgets f&uuml;r Blogs, die sich mit dem Thema Reise befassen (oder allgemein f&uuml;r Blogger, die gerne verreisen). 

Die Widgets sind unabh&auml;ngig voneinander und k&ouml;nnen beliebig in der Sidebar (oder in mehreren Sidebars) platziert werden. Aktuell sind 2 Widgets verf&uuml;gbar:

= Widget: Reise-Countdown =
Mit dem Reisecountdown-Widget kannst du die Zeit bis zu deiner n&auml;chsten Reise runterz&auml;hlen. Das Sidebar-Widget z&auml;hlt die Monate, Wochen, Tage oder sogar Stunden und Sekunden bis zu einem beliebigen Zeitpunkt. Der Countdown wird live aktualisiert (jQuery), so kannst du deine Reise-Vorfreude eindrucksvoll mit deinen Lesern teilen. Alternativ kannst du mit dem Widget auch die Dauer deiner Reise hochz&auml;hlen. So kannst du zum Beispiel zeigen, wie lange du dich schon auf Weltreise befindest.
Reise-Vorfreude ist die sch&ouml;nste Freude! Lass deine Blog-Leser daran teilhaben! 

= Widget: Reise-Schn&auml;ppchen =
Das Reiseschn&auml;ppchen-Widget zeigt aktuelle Gutscheincodes und Rabatte f&uuml;r g&uuml;nstiges Reisen in deiner Sidebar. Du kannst selbst w&auml;hlen, was genau du anzeigen willst (zum Beispiel nur Gutscheine oder nur Reisetipps). Die Daten werden bereitgestellt mit freundlicher Genehmigung von reisetiger.net. Es erscheinen nur Inhalte mit einem echten Mehrwert f&uuml;r deine Leser. Die Daten werden t&auml;glich automatisch aktualisiert. So tut sich immer was auf deinem Blog, auch wenn du mal eine Weile keine neuen Beitr&auml;ge schreiben kannst.

= Reiseblog Plugin Features =
Das Plugin ist schlank und ressourcenschonend und funktioniert in jeder Umgebung.

Nach Aktivierung des Plugins kannst du die Reiseblog-Widgets &uuml;ber Design -&gt; Widgets in deiner Sidebar platzieren.

== Installation ==
= Der einfachste Weg: =
1. Gehe in deinem Wordpress Backend auf Plugins -&gt; Installieren und suche dort nach &quot;Reiseblog&quot;. 
2. Klicke in der Trefferliste bei &quot;Reiseblog Widgets&quot; auf &quot;Jetzt installieren&quot;
3. Aktiviere das Plugin
4. Gehe zu Design -&gt; Widgets und platziere die Reiseblog-Widgets in deiner Sidebar 

= Der manuelle Weg: =
1. Lade das Plugin herunter 
2. Entpacke die ZIP-Datei 
3. Lade den gesamten Ordner `reiseblog-widgets` per FTP in das Verzeichnis `/wp-content/plugins/` auf deinen Blog hoch
4. Gehe in deinem Wordpress Backend zu Plugins und aktiviere das Plugin
5. Gehe zu Design -&gt; Widgets und platziere die Reiseblog-Widgets in deiner Sidebar
